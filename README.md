I focus on improving the stability and accuracy of short-term PV forecasting under highly volatile generation conditions.

### Key Work:
- Developed hybrid forecasting models (GRNN, XGBoost, CNN-BiLSTM)
- Applied bio-inspired optimization algorithms (GWO, WOA, ABC)
- Designed volatility-aware adaptive model structures
- Implemented k-fold validation and robust performance evaluation (MAE, RMSE, MAPE)
